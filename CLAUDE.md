# Schoolasium Design System Platform

## Project Overview

This project is a modern enterprise-grade Design System platform for Schoolasium.

The platform is inspired by:

* Shopify Polaris
* IBM Carbon
* Material UI
* Adobe Spectrum
* Atlassian Design System
* Vercel
* Linear
* Figma documentation systems

This is NOT just a docs website.

This is a centralized scalable ecosystem for:

* design assets
* UI components
* tokens
* branding
* internal employee resources
* AI usage guidelines
* reusable frontend standards
* engineering collaboration

The platform must feel:

* premium
* modern
* scalable
* highly polished
* enterprise-ready

The final product should look like a real company-grade internal design platform.

# Authentication Rules

Employees CANNOT create accounts.

Only admins can:

* create employees
* assign credentials
* manage permissions

Employees only login using provided credentials.
