 Quick Start

Drop it into your project.

Two files do 90% of the work. Load colors_and_type.css for the token foundation, and either wire up tailwind.config.js for class-based usage, or reference tokens directly via var(--*). 

<!-- 1. Load tokens once at the top of your app -->
<link rel="stylesheet" href="colors_and_type.css" />

/* 2. Use tokens directly via CSS custom properties */
.cta {
  background: var(--color-primary-500);
  color:      var(--color-neutral-950);
  border-radius: var(--radius-md);
  box-shadow: var(--shadow-glow-primary);
}

// — or — Tailwind: import the shipped config
module.exports = { presets: ['./tailwind.config.js'] };

 Rules

Logo Do's & Don'ts

Keep the mark legible, energetic, and trustworthy.

 Do
Place on solid dark (#0A0E1A) or solid white. Always preserve clear space equal to the "S" cap-height.
 Do
Use the square mark below 80px width, in app tiles, or any 1:1 surface.
 Don't
Never place the yellow wordmark on yellow or low-contrast backgrounds — it disappears.
 Don't
Don't rotate, stretch, recolor, or add drop shadow. Use --shadow-glow-primary if depth is needed.
